FitBuddy is a web-based application that uses AI to generate personalized workout plans and nutrition tips based on a user’s fitness goals, such as weight loss, muscle gain, or general wellness. Built with FastAPI and integrated with Google’s Gemini AI models, FitBuddy simplifies the process of creating structured and effective fitness routines.

The application takes in user information—like age, weight, goal, and preferred workout intensity—and generates a customized 7-day fitness plan. Users can also submit feedback to update and refine their plan using AI. A complementary nutrition or recovery tip is also provided based on their selected goal.

With a clean user interface and database support via SQLite, FitBuddy ensures each user's data and generated plan are stored and accessible. It offers both API endpoints and an interactive HTML interface for ease of use.
